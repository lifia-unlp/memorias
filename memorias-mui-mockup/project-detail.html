R&D Project Abstract

Distributed Ledgers and Edge Analytics for Secured Industrial Internet of Things

This project aims to resolve severe security and scalability bottlenecks in modern Smart Manufacturing and Industrial IoT (IIoT) setups. By creating localized micro-ledgers, we support high-frequency transactions with absolute cryptographic verification without burdening the central enterprise nodes.

Furthermore, the project designs customized, low-footprint Deep Learning execution layers that reside directly on edge gate-controllers. These layers perform real-time diagnostic checks and predict mechanical failures locally, sharing only consolidated alerts and high-level training weights back to the system core.
